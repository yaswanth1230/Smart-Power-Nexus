# ⚡ Smart Power Nexus using Embedded System

An embedded systems-based intelligent power distribution and control system designed to provide safe, controlled, and efficient electrical power to multiple devices such as mobile phones, electric vehicles, and household appliances.

The project uses a **NodeMCU (ESP8266)** microcontroller, **IR Sensor**, and **Relay Module** to automate power distribution while ensuring electrical safety and efficient energy management. :contentReference[oaicite:0]{index=0}

---

## 📌 Project Overview

Smart Power Nexus is a centralized smart power distribution system that controls electrical power delivery using an embedded controller. The system automatically switches AC power based on sensor input, making it suitable for public charging stations, educational institutions, commercial buildings, and smart infrastructure. :contentReference[oaicite:1]{index=1}

## ⚙ Working Principle

1. Power is supplied to the NodeMCU.
2. The IR Sensor continuously monitors for an object.
3. When an object is detected, the sensor sends a signal to the NodeMCU.
4. The NodeMCU activates the Relay Module.
5. AC Power is supplied to the connected device.
6. When the trigger is removed, the relay disconnects the power supply automatically.
